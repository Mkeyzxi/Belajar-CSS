    Teks soal

    Jelaskan perbedaan dari :

    1. Author style
    2. Reader style
    3. User-agent style

    1. Author style adalah style yang didefinisikan dan diberikan sebuah pendahuluan yang pertama dan akan  diberikan deklarasi selector menurut spesifynya atau bobotnya seperti !important, !important akan didahulukan dalam pemberian warna.

    2. Reader style adalah pengecakan yang dilakukan oleh browser yang mencari jika tidak ada Author seperti pendefinisian !important, maka akan dicari berdasarkan spesify atau bobotnya bisa berupa inline style, id, class, dan element selector. untuk bobotnya sendiri inline style 1,0,0,0, id = 0,1,0,0, class = 0,0,1,0, dan element = 0,0,0,1

    3, user agent style merupakan pengecakan jika tak ada author style dan reader style maka user agent style yang akan diberikan seperti tag universal berupa *.

    Teks soal

    Sebutkan dan jelaskan unit satuan yang ada pada CSS

    Unit pada CSS adalah sebuah style yang ingin didefinisakan dari selector yang diberikan dan dideklrassikan menggunakan sebuah properti, dan value. properti pada css unit ada banyak, seperti Animation, Transition, Box-Shadow, dan lain lain, sedangkan value mendekripsikan sesuatu, atau nilai yang ada pada sebuah properti, seperti selector element h1, ingin diberikan sebuah deklaration berupa property font-family dan memiliki value arial,sans-serif, serta warnanya ingin dirubah menggunakan properti color: dan valuenya warna seperti black, atau bisa juga rgb,rgba(),hsla(), bahkan #fff. untuk properti box-shadow juga misalkan, dia hanya punya value yaitu = inset, jarak x, jarak y, color, blur.
